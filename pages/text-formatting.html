Text Formatting

Text Formatting adalah pemberian format yang mengkhusus pada text. Elemen format dirancang untuk menampilkan jenis teks khusus. Fungsi dari pemformatan text ini adalah untuk memberikan kesan dan tampilan yang berbeda kepada User.

Contoh text formatting:

<b> : Berfungsi untuk mencetak text tebal Contoh : Bold (Teks Tebal)
<Strong> : Berfungsi untuk mencetak text penting Contoh : Strong (Text Penting)
<i> : Berfungsi untuk mencetak text miring Contoh : Italic (Text Miring)
<em> : Berfungsi untuk menekankan text Contoh : em (Text ditekankan)
<mark> : Berfungsi untuk menandai text Contoh : mark (Text ditandai)
<small> : Berfungsi untuk mengecilkan text Contoh : small (Text kecil)
<del> : Berfungsi untuk mencetak text dicoret Contoh : del (text dicoret)
<ins> : Berfungsi untuk mencetak teks sisipan Contoh : ins (Teks sisipan)
<sub> : Berfungsi untuk mencetak teks subscript Contoh : subscript (Teks subscript)
<sup> : Berfungsi untuk mencetak teks superscript Contoh : ins (Teks superscript)
